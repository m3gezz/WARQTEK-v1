A **student documentation platform** built with **Laravel API + React**, designed for **OFPT students**, featuring:

* **Request documents online** without visiting administration offices
* **Role-based access** for students and administrators
* **Real-time request updates**
* **Frontend built with React**
* **Shadcn UI components + TailwindCSS** for a clean, responsive interface
* **Server-side validation** to ensure proper request handling
